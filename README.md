## Instructions:
Requrements:
* Ubuntu 18.04
* ROS Melodic  (other versions are not supported)

Launch rviz with GverilaBot:
```
roslaunch rls_robot_v3_bringup rls_robot_v3_bringup_simulation.launch 
```
